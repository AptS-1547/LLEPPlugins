# EPMEOW Plugin

成为猫娘吧！或者以其他玩家的身份说话   

## Usage:

| 命令                          | 描述                | 执行权限   |
| ----------------------------- | ------------------ | ---------- |
| /speak \<Player\> \<Message\> | 以其他玩家的身份说话 | Operator   |
| /neko list                    | 服务器里面有多少猫娘 | Operator   |
| /neko set \<Player\>          | 让天选之子成为猫娘   | Operator   |
| /neko cancel \<Player\>       | 你不再是猫娘了QAQ   | Operator   |
